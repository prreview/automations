# 100 Indian Startup Research — Job Search Pipeline

## Overview
Research on 100 Indian startups hiring for Fullstack Engineer, Frontend Engineer, and Founding Engineer roles.
Each company has a dedicated .md file with company intelligence, hiring signals, and tailored cold outreach drafts.

## Target Profile
- **Roles**: Fullstack Engineer, Frontend Engineer, Founding Engineer
- **Company Size**: 5-300 employees (early-stage, seed to Series B/C)
- **Location**: India (Bangalore, Mumbai, Delhi, Gurgaon, Pune, Hyderabad, Chennai, Remote India)

## File Structure
Each `{company-slug}.md` file contains:
1. Company metadata (description, website, founders, funding)
2. Hiring signal analysis (open roles, job URLs, ways to apply)
3. Three cold outreach drafts:
   - Cold Email (personalized, 3-4 sentences, lowercase body)
   - LinkedIn/InMail message (under 90 words)
   - Twitter DM (2-3 sentences)
