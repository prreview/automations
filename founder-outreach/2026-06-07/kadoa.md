# Company: Kadoa
- **Description**: Kadoa builds the web data layer for finance. The product uses coding agents to generate deterministic scraping and ETL pipelines that pull data from websites, PDFs, images, and spreadsheets, then push clean outputs into customer systems. The site says the platform is used by top hedge funds and asset managers, which points to a B2B revenue model built on enterprise software and data contracts, not a consumer app. Public funding details are thin, so I wouldn't pin a round size or runway on it; what is clear is that the company is already selling into demanding finance customers and keeping the team small and hands-on.
