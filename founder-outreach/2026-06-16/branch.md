# Company: Branch
- **Description**: Branch is a 6-person software team in Vancouver building real-time structural design and fabrication software for AEC. The product split is simple: Branch Detailing has been running inside StructureCraft since 2020, and Branch Concept is the newer commercial product for early design, where teams can test geometry, structure, cost, carbon, and fabrication data before decisions get locked. The public story says Branch has already supported more than 900,000 sqft of built work through StructureCraft, with 250+ projects and over 10M sqft delivered on the construction side. The site does not publish pricing or a funding round, so revenue looks like software sold into AEC and fabrication workflows, while runway is not public. The company also says it is not chasing the short-term VC game, which suggests a longer build plan.
